# uniapp 聊天应用

这是一个基于 uniapp 开发的聊天应用示例，包含基础的聊天功能和电话拨号功能。

## 功能特点

- 实时聊天功能，支持文本、图片和语音消息
- 表情选择面板
- 语音录制和播放
- 图片预览功能
- 电话拨号功能（顶部按钮）
- 模拟数据支持，方便开发测试

## 技术栈

- uniapp 框架
- Vue.js
- JavaScript
- CSS

## 安装和运行

1. 克隆或下载项目
2. 使用 HBuilderX 或其他 uniapp 开发工具打开项目
3. 运行到指定平台（微信小程序、H5、App 等）

## 主要功能模块

### 聊天界面

- 支持多种消息类型（文本、图片、语音）
- 实时时间显示
- 消息气泡样式区分

### 输入功能

- 文本输入
- 图片选择
- 语音录制
- 表情选择

### 电话功能

- 顶部拨号按钮，支持拨打电话
- 兼容性处理，适配不同环境

## 目录结构

```
my-uniapp/
├── common/             # 公共数据和工具
├── components/         # 组件
├── pages/              # 页面
│   ├── chat/           # 聊天页面
│   ├── chat-list/      # 聊天列表
│   ├── contacts/       # 联系人
│   ├── index/          # 首页
│   └── me/             # 个人中心
├── static/             # 静态资源
├── App.vue             # 入口组件
├── main.js             # 入口文件
├── manifest.json       # 配置文件
└── pages.json          # 页面配置
```

## 开发说明

- 使用 uniapp 框架开发，支持多端发布
- 模拟数据存储在 common/chat-data.js 中
- 电话功能使用 uni.makePhoneCall API
- 所有功能模块都有完善的错误处理和兼容性处理
